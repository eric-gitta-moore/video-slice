📖 API 使用文档

videoSlice(options) 参数说明：

  • video: HTMLVideoElement - 视频元素（可选）
  • width: number | string - 视频宽度
  • height: number | string - 视频高度
  • file: File - 视频文件
  • intervalTime: number - 切片间隔时间（毫秒，默认1000）
  • muted: boolean - 是否静音（默认true）
  • success: () => void - 加载成功回调
  • fail: (message: string) => void - 失败回调
  • backImgFn: (base64: string) => void - 切片回调
  • finish: () => void - 完成回调

返回对象方法：

  • start(): 开始自动切片
  • fastStart(interval?): 快速切片
  • setCurrentTime(time, isSetTime?): 设置/截取指定时间
  • setIntervalTime(time): 设置间隔时间
  • getImgs(): 获取所有切片图片
  • getSrc(): 获取视频源地址
  • destroy(): 销毁实例
  • FileToBase64(file): 文件转Base64

代码示例：

// 导入库 import { videoSlice } from './dist/index.mjs'; // 基础使用 const slicer = videoSlice({ file: videoFile, width: 640, height: 360, intervalTime: 1000, muted: true, success: () => { console.log('视频加载成功'); }, backImgFn: (base64Image) => { console.log('获取到切片:', base64Image); // 显示图片 displayImage(base64Image); }, finish: () => { console.log('切片完成'); }, fail: (error) => { console.error('处理失败:', error); } }); // 开始切片 slicer.start(); // 快速切片（500ms间隔） slicer.fastStart(500); // 获取第10秒的截图 slicer.setCurrentTime(10, true); // 获取所有图片 const allImages = slicer.getImgs(); // 销毁实例 slicer.destroy();
